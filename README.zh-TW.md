# Pocket Trip - 旅費隨記 ✈️💰

這是一個專為旅行者設計的輕量級預算與記帳網頁應用程式。幫助您在旅途中輕鬆記錄開銷、即時換算匯率，並隨時掌握預算狀況，避免旅費超支。

## ✨ 主要功能

### 1. 靈活的記帳模式
*   **🛒 購物車模式 (Cart Mode)**：
    *   專為購物場景設計（如藥妝店、超市）。
    *   可將多樣商品加入「暫存清單」，支援輸入數量與單價。
    *   即時預覽台幣總額，確認無誤後再一次性結帳。
*   **📝 直接記帳 (Direct Entry)**：
    *   適合快速記錄單筆消費（如餐廳用餐、交通票券）。
    *   操作簡單快速，一鍵完成記錄。

### 2. 聰明的匯率與稅務計算
*   **多國貨幣支援**：內建日幣 (JPY)、美金 (USD)、歐元 (EUR) 等多種主流貨幣。
*   **匯率設定**：
    *   🔄 **即時匯率**：支援串接 API 獲取最新市場匯率。
    *   ✍️ **手動匯率**：可依照您實際換匯的匯率進行設定。
*   **稅務計算**：可設定當地消費稅率（如日本 10%），一鍵切換是否含稅，自動計算最終金額。

### 3. 預算與統計
*   **預算監控**：設定旅費總預算，透過視覺化進度條即時查看使用比例。
*   **超支預警**：當花費超過預算時，會顯示醒目的警告提示。
*   **消費分析**：自動計算「每日平均花費」與「最大開支類別」，讓您了解錢都花去哪了。

### 4. 貼心設計
*   **多語言介面**：支援繁體中文與英文介面切換。
*   **資料持久化**：所有資料自動儲存在瀏覽器 (LocalStorage)，關閉視窗或重新整理資料都不會遺失。
*   **PWA 支援**：支援 Progressive Web App，可將網頁「加入主畫面」，像 App 一樣在手機上全螢幕使用。

## 🛠️ 技術棧 (Tech Stack)

本專案採用純前端技術開發，輕量且無需後端伺服器：

*   **HTML5**: 語意化標籤結構。
*   **CSS3**: 使用 **Tailwind CSS** (CDN) 進行快速且現代化的樣式開發。
*   **JavaScript (Vanilla JS)**: 原生 ES6+ 語法，無依賴大型框架 (Vue/React)，保持輕量與高效。

## 🚀 如何使用 (Getting Started)

由於本專案是純靜態網頁，您不需要安裝 Node.js 或任何資料庫。

1.  **下載專案**：
    ```bash
    git clone https://github.com/yourusername/travel-budget-helper.git
    ```
    或者直接下載 ZIP 檔案並解壓縮。

2.  **開啟應用**：
    *   找到資料夾中的 `index.html` 檔案。
    *   直接雙擊用瀏覽器 (Chrome, Safari, Edge 等) 開啟即可使用。
    *   或者使用 VS Code 的 "Live Server" 套件開啟以獲得最佳體驗。

## 📱 手機安裝教學 (PWA)

為了在旅途中方便使用，建議將其安裝到手機：

*   **iOS (Safari)**:
    1. 用 Safari 開啟網頁。
    2. 點擊下方的「分享」按鈕 (Share)。
    3. 往下滑找到並點擊「加入主畫面」 (Add to Home Screen)。

*   **Android (Chrome)**:
    1. 用 Chrome 開啟網頁。
    2. 點擊右上角選單圖示。
    3. 點擊「安裝應用程式」或「加到主畫面」。
